Un clon web del clásico "Super Mario Bros" (1985) construido puramente con HTML5 Canvas, CSS3 y Vanilla JavaScript. 

**Zero dependencias**. Sin bundlers, sin frameworks, sin `npm install` masivos. Solo código nativo corriendo directo en el navegador aprovechando `requestAnimationFrame` para un gameplay fluido a 60fps.

## 🎮 Keybindings / Controles

El juego detecta automáticamente el input del teclado. Para dispositivos táctiles, la UI inyecta un D-Pad y botones virtuales en el DOM.

| Acción | Key |
| :--- | :--- |
| **Moverse** | `Arrow Keys` / `W A S D` |
| **Saltar** | `Space` / `Z` |
| **Sprint / Fuego** | `X` |
| **Start / Pause** | `Enter` |
